## Why gh-infra?

<CardGrid>
  <Card title="YAML instead of HCL" icon="document">
    Declare what your repos should look like in plain YAML.
  </Card>
  <Card title="No state file" icon="approve-check">

  </Card>
  <Card title="Plan before apply" icon="list-format">

  </Card>
